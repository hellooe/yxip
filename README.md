# 🌐 优选IP节点生成器

一个纯前端、无依赖的在线工具，[https://hellooe.github.io/yxip/](https://hellooe.github.io/yxip/)，快速生成优选 ip 节点。支持 Vmess 和 Vless 协议，适配 IPv4、IPv6。

---

## 📦 使用说明

- **协议支持**：Vmess（Base64 编码 JSON）、Vless（标准 URL 格式）。
- **去重机制**：自动对 IP 进行去重，且不改变顺序。
- **性能优化**：即使处理数万条优选 IP 数据，也不会卡死。
- **隐私安全**：所有数据仅在本地处理，无任何网络请求，确保敏感节点信息不会泄露。
